"Wrapper classes are used to convert any primitive type into an object.The primitive data types are not objects, they do not belong to any class, they are defined in the language itself."While storing in data structures which support only objects, it is required to convert the primitive type to object first, so we go for wrapper class.Wrapper classes are used instead of primitive types when an Object is expected."Type wrapper are classes that encapsulate a primitive type within an object."Wrapper classes are used to convert any data type into an object.
