# Implement
## Pipeline
to perform lexrank algorithm, run
```
lexrank_me.py
```
the resource file in contained in
```
data
```
and the output is stored in 
```
result
```
you can use 
```
rouge_evaluate.py
```
to evaluate the performance of Lexrank.
