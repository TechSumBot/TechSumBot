"Java uses "sorted collection" to mean a collection such as SortedSet, where (unlike List), the order that the iterator traverses the collection is in accordance with a specified Comparator or the natural order of the elements.A sorted collection means that not only does the collection have order, but the order depends on the value of the element."A sorted collection is an ordered collection when the order additionally depends on the value of the element to be inserted in, throughout the use of the Comparable interface which provides you with a method to define the sorting criteria.An ordered collection means that the elements of the collection have a specific order."Order collection
